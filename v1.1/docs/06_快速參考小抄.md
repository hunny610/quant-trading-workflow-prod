# 交易提示字自動化系統 - 快速參考小抄

## 🚀 30 秒快速開始

### 最快方式（推薦）
直接告訴 AI：
```
幫我用混合模式分析今天的市場，使用「prompts/混合模式_穩健版.md」
```

### 次快方式
```powershell
# 1. 網路搜索（直接跟 AI 說）
"幫我搜索最新 BTC/ETH 價格和市場新聞"

# 2. 【您手動執行】選擇性抓取 Blave 數據（如果需要深度數據）
# 您可以選擇以下任一方式：
#
# 選項 A：批次抓取（快速，但可能失敗）
# .\scripts\抓取數據腳本\blave_all.ps1
#
# 選項 B：個別任務執行（推薦，更穩定）
npm run capture:blave:toptrader:position
npm run capture:blave:holder_concentration:overview
npm run capture:blave:whale_hunter:overview
#
# 選項 C：跳過 Blave 抓取，只用網路搜索
# （直接進行分析）
```

## 📊 輸出檔案說明

Blave 數據輸出至 `data/blave/` 目錄：
```
data/
├── blave/                          # Blave 抓取腳本輸出
│   └── blave_top_trader_position_YYYY-MM-DD_HHMMSS/
│       ├── blave_top_trader_position_top5.json  ← 看這個！Top 5 多空
│       ├── blave_top_trader_position_parsed.json  ← 解析後數據
│       └── page_full.png  ← 截圖
└── search_out/                     # 00_執行步驟計劃書.md 執行後的查詢結果
```

## ⚠️ 常見問題

### Q: Blave 登入失敗？
A: 手動打開瀏覽器登入 Blave，再執行腳本

### Q: 中文路徑錯誤？
A: 用根目錄的腳本，不要進入中文子目錄

### Q: 數據不夠新？
A: **優先用網路搜索**，Blave 當補充

### Q: 忘記命令了？
A: 看這個小抄，或打開 `docs/05_完整工作流程指南.md`
