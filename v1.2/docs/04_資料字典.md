## 專案概覽

本系統自動抓取 Blave 平台的深度結構化數據，用於量化交易分析。所有數據輸出至 `data/blave/` 目錄，每次執行建立新的時間戳子目錄。

---

## 輸出目錄結構

```
data/
├── blave/                          # Blave 抓取腳本輸出（每次清空）
│   └── {prefix}_{YYYY-MM-DD_HHmmss}/
│       ├── task_result.json              # 任務執行結果
│       ├── summary.json                   # 執行摘要
│       ├── page_full.png                  # 完整頁面截圖
│       ├── page.html                      # 頁面 HTML 原始碼
│       ├── page_url.txt                   # 當前頁面 URL
│       ├── cookies.json                   # 瀏覽器 Cookie（不含敏感資訊）
│       ├── storage.json                   # LocalStorage/SessionStorage
│       ├── tables.json                    # 頁面表格擷取
│       ├── feature_map_blave.json         # Blave 所有可用圖表映射表
│       ├── {task_name}_raw.json           # 原始 API 回應
│       ├── {task_name}_parsed.json        # 解析後標準化數據
│       ├── {task_name}_top5.json          # Top 5 篩選結果（僅部分任務）
│       ├── {task_name}_top10.json         # Top 10 篩選結果（僅部分任務）
│       └── {task_name}_capture.json       # 擷取失敗時的錯誤記錄
├── coinclass/                      # CoinGlass 抓取腳本輸出
│   └── {prefix}_{YYYY-MM-DD_HHmmss}/
│       ├── task_result.json              # 任務執行結果
│       ├── summary.json                   # 執行摘要
│       ├── page_full.png                  # 完整頁面截圖
│       ├── page.html                      # 頁面 HTML 原始碼
│       ├── page_url.txt                   # 當前頁面 URL
│       ├── cookies.json                   # 瀏覽器 Cookie（不含敏感資訊）
│       ├── storage.json                   # LocalStorage/SessionStorage
│       ├── tables.json                    # 頁面表格擷取
│       ├── feature_map_blave.json         # Blave 所有可用圖表映射表
│       ├── feature_map_coinglass.json     # CoinGlass 所有可用圖表映射表
│       ├── coinglass_hyperliquid_whales.json  # Hyperliquid Whale Tracker 數據
│       └── coinglass_liquidation_heatmap_model3_responses.json  # Liquidation Heatmap 數據
└── search_out/                     # 手動保存的搜索結果和分析報告
```

### 手動保存搜索結果（可選）

如果需要保存網路搜索結果，可手動保存到 `data/search_out/` 目錄：

1. 使用 `data/search_out/範例_搜索結果保存格式_2026-05-05.md` 作為模板
2. 複製模板文件，重命名為您的標題（如：`BTC價格查詢_2026-05-05.md`）
3. 將搜索結果貼上並保存

**文件名格式建議**：`{標題}_{YYYY-MM-DD_HHmmss}.md`

## Blave 任務輸出檔案說明

### 1. blave:toptrader:position（Blave頂尖交易員-部位）

| 檔案名稱 | 說明 |
|---------|------|
| `blave_top_trader_position_raw.json` | 原始 API 回應 |
| `blave_top_trader_position_parsed.json` | 標準化後的多空部位列表 |
| `blave_top_trader_position_top5.json` | Top 5 多/空部位 |

#### blave_top_trader_position_parsed.json 結構

```json
{
  "ok": true,
  "capturedAtUrl": "https://blave.org/studio/zh/charts/blave_top_trader/position",
  "long": [
    {
      "token": "BTC",
      "tokenId": 1,
      "price": 61234.56,
      "entry": 60000.00,
      "position": 2.5
    }
  ],
  "short": [
    {
      "token": "ETH",
      "tokenId": 2,
      "price": 3456.78,
      "entry": 3500.00,
      "position": -15.3
    }
  ]
}
```

| 欄位 | 型別 | 說明 |
|-----|------|------|
| `ok` | boolean | 擷取是否成功 |
| `capturedAtUrl` | string | 擷取時的頁面 URL |
| `long[].token` | string | 代幣符號 |
| `long[].tokenId` | number | 代幣 ID |
| `long[].price` | number | 當前價格 |
| `long[].entry` | number | 入場價 |
| `long[].position` | number | 部位大小（正數=多頭，負數=空頭） |

#### blave_top_trader_position_top5.json 結構

```json
{
  "ok": true,
  "top5Long": [
    {
      "token": "BTC",
      "tokenId": 1,
      "price": 61234.56,
      "entry": 60000.00,
      "position": 2.5
    }
  ],
  "top5Short": [
    {
      "token": "ETH",
      "tokenId": 2,
      "price": 3456.78,
      "entry": 3500.00,
      "position": -15.3
    }
  ]
}
```

### 2. Overview 類任務（共 9 個）

以下任務共用相同的輸出格式：
- `blave:holder_concentration:overview`（籌碼集中度-總覽）
- `blave:whale_hunter:overview`（巨鯨警報-總覽）
- `blave:market_sentiment:overview`（市場情緒-總覽）
- `blave:taker_intensity:overview`（多空力道-總覽）
- `blave:liquidation:overview`（爆倉-總覽）
- `blave:funding_rate:overview`（資金費率-總覽）
- `blave:funding_rate:history`（資金費率-歷史）
- `blave:oi_imbalance:overview`（OI失衡-總覽）
- `blave:squeeze_momentum:history`（擠壓動能-歷史）

| 檔案名稱 | 說明 |
|---------|------|
| `{task_name}_raw.json` | 原始 API 回應 |
| `{task_name}_parsed.json` | 標準化後的代幣列表 |
| `{task_name}_top10.json` | Top 10 依 Alpha 絕對值排序 |

#### {task_name}_parsed.json 結構

```json
{
  "ok": true,
  "capturedAtUrl": "https://blave.org/studio/zh/charts/xxx/overview",
  "overallAlpha": 1.23,
  "overallScore": 85.5,
  "symbols": [
    {
      "token": "BTC",
      "tokenId": 1,
      "price": 61234.56,
      "pctChange": 2.34,
      "chg24h": 3.45,
      "alpha": 1.23
    }
  ]
}
```

| 欄位 | 型別 | 說明 |
|-----|------|------|
| `ok` | boolean | 擷取是否成功 |
| `capturedAtUrl` | string | 擷取時的頁面 URL |
| `overallAlpha` | number | 整體 Alpha 值 |
| `overallScore` | number | 整體分數 |
| `symbols[].token` | string | 代幣符號 |
| `symbols[].tokenId` | number | 代幣 ID |
| `symbols[].price` | number | 當前價格 |
| `symbols[].pctChange` | number | 漲跌幅（%） |
| `symbols[].chg24h` | number | 24h 漲跌幅（%） |
| `symbols[].alpha` | number | Alpha 值 |

#### {task_name}_top10.json 結構

```json
{
  "ok": true,
  "top10": [
    {
      "token": "SOL",
      "tokenId": 3,
      "price": 145.67,
      "pctChange": 5.67,
      "chg24h": 6.78,
      "alpha": 3.45
    }
  ]
}
```

> **排序規則**：依 `alpha` 絕對值由大到小排序，取前 10 名

## 通用檔案格式

### task_result.json

```json
{
  "task": "blave:toptrader:position",
  "ok": true,
  "top5LongCount": 5,
  "top5ShortCount": 5
}
```

| 欄位 | 型別 | 說明 |
|-----|------|------|
| `task` | string | 執行的任務名稱 |
| `ok` | boolean | 任務是否成功 |
| `top5LongCount` | number | Top 5 多頭數量（僅 toptrader:position） |
| `top5ShortCount` | number | Top 5 空頭數量（僅 toptrader:position） |
| `symbolCount` | number | 代幣總數（僅 overview 類任務） |
| `top10Count` | number | Top 10 數量（僅 overview 類任務） |

---

### summary.json

```json
{
  "stamp": "2026-05-05_143022",
  "urlRequested": "https://blave.org/studio/zh/charts/blave_top_trader/position",
  "finalUrl": "https://blave.org/studio/zh/charts/blave_top_trader/position",
  "outDir": "data/blave/blave_top_trader_position_2026-05-05_143022",
  "profileDir": ".profile",
  "channel": "msedge"
}
```

| 欄位 | 型別 | 說明 |
|-----|------|------|
| `stamp` | string | 時間戳記（YYYY-MM-DD_HHmmss） |
| `urlRequested` | string | 請求的 URL |
| `finalUrl` | string | 最終抵達的 URL |
| `outDir` | string | 輸出目錄路徑 |
| `profileDir` | string | 瀏覽器設定檔目錄 |
| `channel` | string | 瀏覽器頻道（msedge/chrome） |

### feature_map_blave.json

Blave 平台所有可用圖表的完整映射表，包含 30+ 個圖表。

```json
[
  {
    "key": "holder_concentration/overview",
    "name": "籌碼集中度-總覽",
    "url": "https://blave.org/studio/zh/charts/holder_concentration/overview",
    "api": "https://api.blave.org/studio/charts/holder_concentration/overview"
  }
]
```

| 欄位 | 型別 | 說明 |
|-----|------|------|
| `key` | string | 圖表唯一識別碼 |
| `name` | string | 圖表中文名稱 |
| `url` | string | 圖表頁面 URL |
| `api` | string | API 端點（若無則為 null） |

## 資料型別定義

### SymbolItem（代幣項目）

| 欄位 | 型別 | 必填 | 說明 |
|-----|------|------|------|
| `token` | string | 是 | 代幣符號（如 BTC、ETH） |
| `tokenId` | number | 否 | Blave 內部代幣 ID |
| `price` | number | 否 | 當前價格 |
| `pctChange` | number | 否 | 漲跌幅（%） |
| `chg24h` | number | 否 | 24h 漲跌幅（%） |
| `alpha` | number | 否 | Alpha 值 |

---

### PositionItem（部位項目）

| 欄位 | 型別 | 必填 | 說明 |
|-----|------|------|------|
| `token` | string | 是 | 代幣符號 |
| `tokenId` | number | 否 | Blave 內部代幣 ID |
| `price` | number | 否 | 當前價格 |
| `entry` | number | 否 | 入場價 |
| `position` | number | 否 | 部位大小（正=多，負=空） |
