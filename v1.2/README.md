# 交易提示字自動化系統

## 🚀 快速開始

```powershell
# 一鍵啟動（推薦）
npm run go

# 或手動執行
npm run clear          # 清空舊數據
npm run high           # 抓取高優先級3項數據
npm run save:report    # 保存分析報告
```

## 📋 常用命令

| 動作 | 命令 |
|-----|------|
| 一鍵啟動 | `npm run go` |
| 清空數據 | `npm run clear` |
| 高優先級抓取 | `npm run high` |
| 保存報告 | `npm run save:report` |
